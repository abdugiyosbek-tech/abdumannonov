# Changelog

## 3.0.0 MVP

- 30 kunlik tracking bugungi kundan boshlab siljib boradigan qilindi.
- Har bir kunning mashg‘ulot rejasi jadval ichida qo‘lda tahrirlanib saqlanadi.
- Dam/tiklanish kunlarida faollik sport foizi yoki qadam foizining kattasi orqali baholanadi.
- Haftalik hisobot bo‘sh kunlarni ma’lumot sifatida qabul qilmaydi.
- Portal va Excel uchun default baholash vaznlari 40/35/15/10 ko‘rinishida birlashtirildi.
- Fotolar LocalStorage o‘rniga IndexedDB bazasiga siqilgan holda saqlanadi.
- Backup importiga versiya, tuzilma, sana, qiymat va foto validatsiyasi qo‘shildi.
- Foydalanuvchi matnlari DOMga `textContent` orqali chiqarilib, XSS xavfi kamaytirildi.
- Qora va oq rejim qo‘shildi; tanlov brauzerda saqlanadi.
- Kod `config.js`, `core.js`, `storage.js`, `app.js` va `styles.css` fayllariga ajratildi.
- PWA cache yangilash, eski cachelarni o‘chirish va offline fallback yaxshilandi.
- GitHub Pages uchun `.nojekyll`, README, testlar va paket metadata qo‘shildi.
